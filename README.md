### 构架

服务器: 阿里云(2CUP,4G内存,100M峰值带宽，ssd硬盘20G+40G) + 阿里云(OSS+CDN) + 阿里云(RDS/mysql)

后台: (docker -> Iojs + pm2 + mysql/mongodb)

最高承载量:  1035417pv/日 (应该不是瓶颈)

统计工具: tongji.baidu.com (准备全面换google analytics，百度统计还是不太好用)

### 后台
